# baget-device

Файл-устройство (на самом деле это симлинк на fuse), который при каждом прочтении генерирует новый бугурт.

## Установка

```
go get github.com/malyutinegor/baget-device
```

## Запуск

```
> go build
> su -c "./baget-device -path /dev/baget &"
> cat /dev/baget
```
